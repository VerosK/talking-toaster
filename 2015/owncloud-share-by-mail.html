Owncloud 7 - customize mail messages

OwnCloud 7 came packed with lots of new features. Many happy users of OwnCloud uses it as file transfer solution (me including).

The default mail message sent to the user receiving the file is a bit unformal and is considered unpolite. It starts Hey there. How to change the message?

The good news is file is stored in core/templates/mail.php (HTML version) and core/templates/altmail.php (plain text version). mc If you change that templates, the messages can be a bit more polite.
