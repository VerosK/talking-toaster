Jenkins on Windows + FTP Publisher

Pro neziskovku - mateřské centrum - jsem připravil malý program v Pythonu v PySide GUI. Jenže neziskovka má počítače na Windows a můj vývojářský stroj je na Linuxu. A nutit maminky instalovat do náhodného počítače Python pro Windows včetně knihovně je ... politicky neprůchodné(?).

Naštěstí existuje berlička v podobě PyInstaller. Do jednoho počítače s Windows se nainstaluje Python, všechny potřebné balíčky a pyinstaller z Pythonového programu udělá spustitelný .EXE soubor.

Ve skutečnosti je v tom EXE souboru zabalený celý Python včetně závislostí, při každém spuštění se to rozbalí do %TEMPDIR%, ale to jsou skoro nepodstatné detaily. Uživatel vidí .EXE soubor a je spokojený.

I've prepared some small Python GUI program for one non-profit organization. As my development machine runs on Linux and the customer runs Windows machine, I need to package the software for the-other OS.
